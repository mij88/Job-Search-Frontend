# Job-Search-Frontend

A Project to allow users to record and edit jobs that they have applied 

### Run Program
First follow instruction to download and run backend from 
[Job-Search-Backend](https://github.com/mij88/Job-Search-Backend)

Run frontend
```bash
cd frontend
npm start
```
